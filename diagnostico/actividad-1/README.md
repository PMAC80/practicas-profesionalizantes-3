# Gestión de Stock - Planta de Reciclaje

Trabajo práctico introductorio para el desarrollo de un sistema mínimo de gestión de stock.

## 📋 Requisitos

- Node.js
- MySQL
- Navegador web

## 🚀 Cómo ejecutar el proyecto

### 1. Base de datos
- Crear una base de datos en MySQL
- Ejecutar el archivo `schema.sql` para crear la tabla `materiales`

### 2. Backend
```bash
cd backend
npm install
node server.js
